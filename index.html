About Me

Sedikit tentang saya

Saya adalah seorang lulusan baru dari Universitas Muslim Indonesia, jurusan Teknik Informatika. Saya menempuh pendidikan dari tahun 2020 hingga 2024. Selama masa studi, saya memilih Rekayasa Perangkat Lunak sebagai mata kuliah peminatan dan berfokus pada pengembangan aplikasi, khususnya pengembangan web menggunakan Laravel. Saat ini, saya juga sedang mempelajari teknologi web front-end dan back-end dengan menggunakan React.js dan Node.js untuk memperluas keterampilan saya dalam pengembangan aplikasi web.

Selama masa studi, saya aktif mengikuti berbagai pelatihan untuk mengasah keterampilan dan meningkatkan kompetensi saya di bidang teknologi, di antaranya:

  • Program Studi Independen Kampus Merdeka Bangkit Academy, yang diselenggarakan oleh PT. Dicoding Academy dengan alur pembelajaran Cloud Computing.
  • Program Vocational School Graduate Academy (VSGA) Batch 6 Junior Web Developer, yang diselenggarakan oleh Kementerian Komunikasi dan Informatika (Kominfo).

Dengan pengalaman dan pelatihan yang telah saya ikuti, saya sangat bersemangat mengembangkan aplikasi web yang dapat memberikan solusi inovatif untuk memecahkan berbagai tantangan. Saya percaya bahwa teknologi memiliki peran besar dalam menciptakan perubahan positif, dan saya ingin berkontribusi melalui karya yang bermanfaat bagi masyarakat.
